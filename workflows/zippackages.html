Zip Packages¶

This tool takes a folder, usually of HathiTrust packages, zips each subfolder, and copies the resultant tree to a different location. Input is a root folder, usually for a HathiTrust shipment, containing multiple subfolders, each one a HathiTrust digitized item.
Output is a destination location for the newly generated file.
